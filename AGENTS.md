# Project Guidance

この Unity プロジェクトは、VR アプリ開発を学ぶためのチュートリアル用プロジェクトです。

重要方針:

- 効率よりも、安定して着実に前に進むことを優先する。
- 一度に大きく変更せず、1 ステップずつ理解しながら進める。
- 既に動いている Grab、Rigidbody、Collider、Meta XR Building Block の中核部分は、必要がなければ触らない。
- 見た目や演出を追加する場合は、まず子オブジェクトとして追加する。
- 新機能を追加した後は、掴める、投げられる、初期位置に戻る、スコアが増える、という基本動作を確認する。
- 変更前に「触る部分」と「触らない部分」を明確にする。
- 学習日の最終状態をコミット、PUSHする際に、新しいリソースをUnity Asset Storeからインストールしている場合は、README.mdの「外部リソース」にアセット名、Storeリンク、再インポートが必要な旨を追記する。
- 説明は日本語で、学習者向けに段階的に行う。
