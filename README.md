# VR_tutrial
VRアプリの作り方を、Codexを使って勉強中。

プロジェクトは
1. Cube Throw

   - 掴む
   - 投げる
   - 的に当てる
   - スコアをカウントする

など、１ステップずつ、VR内でできることを増やしていく予定。

# 外部リソース
このプロジェクトでは、以下のリソースをUnity Asset Storeからインストールして利用しています。

- [FREE Casual Game SFX Pack](https://marketplace.unity.com/packages/audio/sound-fx/free-casual-game-sfx-pack-54116)
  - プロジェクト内フォルダ名: `Assets/Casual Game Sounds U6`
  - Gitには含めていないため、別環境で利用する場合はUnity Asset Storeから再インポートしてください。

- [FREE Industrial building/Garage](https://assetstore.unity.com/packages/3d/environments/industrial/free-industrial-building-garage-123146)
  - プロジェクト内フォルダ名: `Assets/Industrial building`
  - Gitには含めていないため、別環境で利用する場合はUnity Asset Storeから再インポートしてください。
